## Introdução
Neste projeto em C#, o usuário digita um número ímpar e a aplicação exibe um diamante formado pela letra `X`. A cada execução, é possível gerar um novo diamante ou encerrar o programa.

## Funcionalidades
- **Desenho de Diamante**: Criação de um diamante simétrico baseado no número digitado.
- **Validação de Entrada**: Impede a inserção de letras, números pares, decimais, 0 e 1.
- **Interface Simples no Console**: Interface textual dinâmica para interação direta.
- **Repetição do Processo**: Permite ao usuário criar novos diamantes sem reiniciar o programa.

## Como Usar
1. O sistema solicita ao usuário que digite um número ímpar (mínimo 3).
2. Caso a entrada seja válida, o diamante é desenhado no console.
3. Ao final, é exibida a opção: "Deseja fazer um novo triângulo? (s/n)".
4. O processo pode ser repetido quantas vezes o usuário desejar.
